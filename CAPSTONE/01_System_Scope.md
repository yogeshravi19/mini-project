# 1. System Scope

## 1.1 Project Title
TRUEGUARD: A Comprehensive Framework for Analyzing and Mitigating LLM Hallucinations

## 1.2 Objective
To analyze the phenomenon of Large Language Model (LLM) hallucinations, classify them based on a robust taxonomy, evaluate existing mitigation strategies across 20 research papers, and propose/validate the TRUEGUARD framework as a robust mechanism for hallucination detection and mitigation.

## 1.3 Scope of the Project
- **In-Scope:**
  - Data extraction and comparative analysis of 20 recent research papers on LLM hallucination.
  - Visualization of key performance metrics (AUROC, F1 Score, Speedup, Reduction Rate) using R (ggplot2).
  - Implementation of a generalized TRUEGUARD detection pipeline using API-based evaluations (Google Colab / Cloud environment).
- **Out-of-Scope:**
  - Training or fine-tuning local open-source LLMs natively on edge devices (due to hardware limitations).
  - Production-scale deployment of the framework into an enterprise environment.

## 1.4 Target Audience
Researchers, Data Scientists, and AI Engineers integrating LLMs into applications where factual accuracy, logical consistency, and reliability are critical.
